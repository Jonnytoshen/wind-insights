## 1. 产品概述

**Wind Insights** 是一款面向风电行业专业人员的在线虚拟测风塔分析平台。用户通过 **向导式（Wizard）流程**，在地图上选定候选点位，系统自动调用 **NASA POWER API** 获取 MERRA2 逐小时再分析风速数据，完成多维度风资源评估，并一键生成符合行业规范的 **PDF 分析报告**，替代或辅助传统实地测风工作，显著降低项目前期成本与周期。

### 1.1 产品定位

> "10 分钟完成一个点位的完整风资源评估，并输出可交付的工程报告。"

### 1.2 核心价值

- **低成本替代测风**：无需部署气象仪器，利用公开再分析数据快速评估
- **多维度分析**：覆盖风资源评估所需的全部核心气象统计指标
- **专业报告输出**：PDF 报告可直接用于可研汇报或融资材料
- **多高度对比**：支持在同一点位对不同轮毂高度横向比选

## 2. 目标用户与使用场景

### 2.1 目标用户

| 角色                   | 核心诉求                                       | 使用频率   |
| ---------------------- | ---------------------------------------------- | ---------- |
| 风电开发工程师（选址） | 快速筛查候选场址风资源好坏，支撑前期决策       | 高（日常） |
| 风资源评估顾问         | 生成可交付的评估报告，减少重复性统计工作       | 高（日常） |
| 政府/能源规划机构      | 区域风资源摸底，制定可再生能源发展规划         | 中（专项） |
| 风电投资人/业主        | 独立核验项目风资源，辅助投资决策               | 低（按需） |

### 2.2 核心使用场景

#### 场景 A：单点快速选址分析
用户在开发初期需在地图上圈定若干候选点位，快速获得每个点位的年均风速、风功率密度等概况指标，筛选出值得深入的点位。

**用户旅程**：进入应用 → 在地图上点击候选位置 → 选择评估高度与时间范围 → 查看自动生成的风资源概览卡片 → 对多个点位对比排序

#### 场景 B：长序列深度分析（虚拟测风塔）
用户对筛选后的重点点位进行 10 年以上历史数据的精细化分析，输出与真实测风塔报告等价的统计成果。

**用户旅程**：选定点位 → 配置参数（高度、时间跨度≥10年、数据频率） → 等待数据拉取 → 逐步浏览各分析模块 → 生成 PDF 报告

#### 场景 C：多高度轮毂对比
用户需要为尚未确定机型的项目比较 80 m / 100 m / 120 m / 140 m 等不同高度的风资源，以支撑机型选型。

**用户旅程**：选定点位 → 添加多个高度层 → 系统并行分析并绘制各高度对比图 → 导出包含风切变参数的汇总表

#### 场景 D：批量区域普查（未来扩展）
用户上传多个点位坐标（CSV），系统批量分析并输出汇总排名。（本版本不纳入，记录为待规划功能。）

## 4. 数据基础

### 4.1 数据源：NASA POWER API

| 属性       | 描述                                                                 |
| ---------- | -------------------------------------------------------------------- |
| 数据集     | MERRA-2 再分析数据（Modern-Era Retrospective analysis for Research） |
| API 端点   | `https://power.larc.nasa.gov/api/temporal/hourly/point`             |
| 空间分辨率 | 约 0.5° × 0.625°（经纬度网格，约 50 km）                            |
| 时间分辨率 | 逐小时                                                               |
| 时间范围   | 1981 年至今（滚动更新，约延迟 3 个月）                              |
| 固定高度风速参数 | `WS2M`（2 m）、`WS10M`（10 m）、`WS50M`（50 m）              |
| 自定义高度风速   | 通过 `wind-elevation` 参数（单位：m）直接请求 **10～300 m 任意高度** 的修正风速；API 内部基于 WS50M + 地表粗糙度类型幂律公式计算 |
| 风向参数   | `WD10M`（10 m）、`WD50M`（50 m）；自定义高度暂无独立风向，使用最近固定高度风向代替 |
| 访问方式   | 开放 REST API，无需注册（有速率限制）                                |
| 频率限制   | 每次最大请求时长约 366 天；长序列需分段请求并合并                   |

### 4.2 自定义高度风速获取机制

NASA POWER API 通过 `wind-elevation` 参数支持在 **10 m ～ 300 m 范围内请求任意指定高度的风速**，无需后端自行实现插值逻辑。API 内部采用如下幂律公式从 WS50M 修正计算：

$$
V(h) = V_{50m} \times \left(\frac{h}{50}\right)^{\alpha}
$$

其中 $\alpha$ 由用户通过 `wind-surface` 参数选择**地表覆盖类型**（surface type alias）决定，而非实测拟合，例如：

| 别名           | 地表描述                           |
| -------------- | ---------------------------------- |
| `vegtype_11`   | 裸土（Rough bare soil）            |
| `vegtype_6`    | 稀树草原（Savanna，18 m 乔木 30%） |
| `vegtype_12`   | 农田（Crop，小麦+落叶乔木 10%）    |
| `openwater`    | 开阔水面                           |
| `airportgrass` | 机场草地（平坦粗草）               |

> **影响**：应用无需在后端手动做切变外推，可在请求时直接指定轮毂高度；但用户需选择合理的地表类型以匹配场址实际情况。高度上限为 **300 m**，超出此范围不支持。

风切变指数 $\alpha$ 的**实际拟合值**（用于报告展示）仍通过比较多个请求高度的风速反推获得（见 F5.5）。

### 4.3 数据质量处理

| 问题           | 处理策略                                                     |
| -------------- | ------------------------------------------------------------ |
| 缺失值         | 标记后排除计算，在报告中注明缺失率                           |
| 异常值（粗误差） | 采用 3σ 法剔除（可选，用户可配置）                           |
| 分段请求合并   | 按年分段请求，后端合并为完整时间序列                         |
| API 失败重试   | 指数退避重试，最多 3 次；失败后提示用户                      |

### 5.2 详细功能需求

#### F1 — 向导流程框架（Wizard Shell）

| 步骤 | 名称       | 说明                                                           |
| ---- | ---------- | -------------------------------------------------------------- |
| 1    | 选择点位   | 地图交互选点，确认坐标                                         |
| 2    | 配置参数   | 分析高度、时间范围、参考高度、风切变初始值等                   |
| 3    | 数据加载   | 可视化进度，展示拉取状态                                       |
| 4    | 分析总览   | 展示关键指标摘要卡片                                           |
| 5    | 详细分析   | 分 Tab/Section 展示各分析模块                                  |
| 6    | 报告生成   | 配置报告封面信息，生成 PDF                                     |

- 向导流程支持**前进/后退**导航，已完成步骤可跳转回溯
- 分析完成后的状态需可**保存为草稿**（本地 localStorage，MVP 阶段）

#### F2 — 地图选点模块

- 使用 **Mapbox GL JS** 或 **Leaflet + OpenStreetMap** 作为底图
- 支持地图缩放、平移
- 点击地图任意位置放置测风塔标记（Marker）
- 支持坐标直接输入（经度/纬度），用于精确定位
- 显示 MERRA2 网格背景（可选，透明叠加），帮助用户理解数据分辨率
- 最近选过的点位以历史标记形式展示（本地存储，最多 20 个）

#### F3 — 参数配置模块

| 参数                 | 默认值              | 约束                                                       |
| -------------------- | ------------------- | ---------------------------------------------------------- |
| 分析高度（可多选）   | 100 m               | 自由输入，范围 **10～300 m**；预设快捷值：10/50/80/100/120/140 m |
| 时间范围-起始        | 当前年 - 10 年       | 最早 1981 年                                               |
| 时间范围-结束        | 最近完整年          | 不超过 API 数据截止日                                      |
| 地表覆盖类型         | `vegtype_11`（裸土）| 下拉选择 NASA POWER 支持的 surface type alias；影响 API 内部 α 取值 |
| 异常值过滤           | 开启（3σ）          | 可关闭                                                     |
| 项目名称             | 空                  | 用于报告封面                                               |

#### F4 — 数据拉取与处理引擎

- 前端调用后端 API，后端负责向 NASA POWER API 发起请求（规避 CORS）
- 每个分析高度对应一组独立请求（携带 `wind-elevation=<h>` 及 `wind-surface=<type>`），按**年**批量并行发送，最终合并
- 风向数据：使用最接近的固定高度参数（≤50 m 取 `WD10M`，>50 m 取 `WD50M`）——风向随高度变化有限，此近似在工程评估中普遍接受
- 实时进度反馈（已完成年数 / 总年数 × 高度数）
- 数据预处理：缺失值统计、异常值标记、时区统一（UTC）
- 处理后的数据**缓存**在后端内存/Redis 中，有效期 30 分钟，避免重复请求
- **后端无需自行实现切变外推**，所有指定高度风速直接由 NASA POWER API 返回

#### F5 — 风资源分析与可视化

##### F5.1 基础统计

| 指标                   | 展示形式                              |
| ---------------------- | ------------------------------------- |
| 年均风速               | 数值卡片 + 历年折线图                 |
| 月均风速               | 12 个月柱状图（含标准差区间）         |
| 日变化（小时平均）     | 24 小时折线图                         |
| 风速频率直方图         | 分组柱状图（风速区间 vs. 频率 %）     |
| 风速持续曲线           | 从高到低排列的年持续时数曲线（FDC）   |

##### F5.2 风速/风向玫瑰图

- 标准 16 方位玫瑰图
- 颜色区分风速段（< 3、3-6、6-9、9-12、>12 m/s）
- 支持按季节切换（全年/春/夏/秋/冬）
- 显示主导风向及其频率占比

##### F5.3 Weibull 分布分析

- 采用**最大似然估计（MLE）**拟合 Weibull 形状参数 $k$ 和尺度参数 $c$
- 展示实测频率分布与 Weibull 拟合曲线叠加图
- 输出 $k$、$c$ 参数值及拟合优度（R²）

##### F5.4 风功率密度（WPD）

$$WPD = \frac{1}{2} \rho \overline{v^3} \quad \text{(W/m²)}
$$

- 空气密度 $\rho$ 采用标准大气 1.225 kg/m³ 或根据海拔/温度订正（用户可配置）
- 输出：年均 WPD、月均 WPD 柱状图、IEC 风电场等级判定（Ⅰ/Ⅱ/Ⅲ/S）

##### F5.5 多高度风切变分析

- 对所选各高度层的年均/月均风速绘制风廓线（对数坐标）
- 各高度风速数据均由 NASA POWER API `wind-elevation` 参数直接返回（API 内部使用用户选定的地表类型 α 计算）
- 通过对多个高度实际返回的年均风速进行**幂律回归拟合**，反推场址**等效实际风切变指数 $\alpha_{fit}$**，用于报告展示和机型选型参考
- 注明：$\alpha_{fit}$ 基于 MERRA2 网格尺度数据拟合，与真实测风结果可能存在偏差
- 输出：各高度年均风速汇总表、地表类型对应 α 值、回归拟合 $\alpha_{fit}$

##### F5.6 湍流强度（TI）

$$TI = \frac{\sigma_v}{\bar{v}}$$

- 以 1 m/s 为风速区间分组统计 TI
- 与 IEC 61400-1 标准参考曲线对比（Class A / B / C）
- 折线图展示 TI 随风速变化趋势

##### F5.7 极端风速（50 年一遇）

- 采用 **Gumbel 极值 I 型**或 **GEV 分布**对年最大风速序列拟合
- 外推 50 年重现期风速 $V_{50}$
- 对应 IEC 61400-1 的 $V_{ref}$ 等级判定
- 附带置信区间（90% CI）

##### F5.8 代表年选取分析

从长序列历史数据中筛选出统计特性最接近长期平均水平的单个年份，作为风资源详细评估的代表性数据集（常用于风机发电量计算）。

**选取方法（综合评分法）**：

| 指标               | 权重 | 说明                                                       |
| ------------------ | ---- | ---------------------------------------------------------- |
| 年均风速偏差率     | 60%  | 候选年年均风速相对长期均值的偏差百分率（越小越好）        |
| 月际分布相关性     | 40%  | 候选年月均风速序列与长期月均序列的 Pearson 相关系数（越大越好）|

**输出内容**：

- 代表年年份及其年均风速
- 与长期均值的偏差率（%）
- 月际相关系数
- 各候选年评分排名表（可展开查看）
- 代表年逐月均风速与长期月均对比柱状图

#### F6 — 多高度对比视图

- 当用户选择 ≥ 2 个高度时自动启用
- 汇总对比表（各高度：年均风速、WPD、$k$、$c$、$\alpha$）
- 各高度 Weibull 曲线叠加图
- 各高度风速频率分布叠加图

#### F7 — PDF 报告生成

报告封面信息（用户填写）：
- 项目名称、项目地址（自动填入坐标）、报告日期、编制单位、保密等级

报告结构（详见 §9）。

## 6. 非功能需求

### 6.1 性能

| 场景                             | 目标                           |
| -------------------------------- | ------------------------------ |
| 单点 10 年数据完整分析           | 端到端 ≤ 60 秒                 |
| 单点 20 年数据完整分析           | 端到端 ≤ 120 秒                |
| PDF 报告生成                     | ≤ 30 秒                        |
| 地图交互响应                     | ≤ 200 ms                       |

### 6.2 可用性与可靠性

- NASA POWER API 不可用时，展示友好错误提示，并提供重试按钮
- 前端分析步骤结果本地暂存，页面刷新后可恢复（localStorage）
- 支持现代浏览器：Chrome 100+、Edge 100+、Firefox 100+、Safari 16+

### 6.3 安全性

- 后端作为 NASA POWER API 的代理，避免直接暴露客户端 IP
- 不存储用户坐标或分析结果到持久化数据库（MVP 阶段，无账号体系）
- 所有前后端通信通过 HTTPS
- 后端需对请求参数（坐标范围、时间范围）做服务端校验，防止注入和越界请求

### 6.4 可访问性

- 页面适配 1280 × 800 及以上分辨率
- 图表提供数据表格备用展示（Screen Reader 可访问）
- 支持中文界面（MVP 默认中文，预留国际化接口）

## 7. 技术架构

### 7.1 推荐技术栈

| 层级        | 技术选型                                     | 理由                                           |
| ----------- | -------------------------------------------- | ---------------------------------------------- |
| 前端框架    | **Vue 3 + TypeScript**                       | 组件化、生态完善                               |

### 7.3 核心 API 接口设计

```
POST /api/analysis/start
  Request: { lat, lon, heights[], start_year, end_year, alpha_init, filter_outliers }
  Response: { task_id, estimated_seconds }

GET  /api/analysis/{task_id}/status
  Response: { status, progress, message }

GET  /api/analysis/{task_id}/result
  Response: { summary, statistics, charts_data, ... }

POST /api/report/generate
  Request: { task_id, project_name, organization, confidentiality }
  Response: { pdf_url }  (or stream binary)
```

## 8. 界面与交互设计

### 8.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  Logo  Wind Insights          [语言▾]  [历史记录]  [?]   │
├─────────────────────────────────────────────────────────┤
│  步骤条：① 选择点位 → ② 配置参数 → ③ 加载数据          │
│          → ④ 分析总览 → ⑤ 详细分析 → ⑥ 生成报告        │
├─────────────────────────────────────────────────────────┤
│                                                          │
│              [当前步骤内容区]                             │
│                                                          │
├─────────────────────────────────────────────────────────┤
│  [← 上一步]                              [下一步 →]      │
└─────────────────────────────────────────────────────────┘
```

### 8.2 步骤详细交互

**步骤 1 — 选择点位**
- 全屏地图，上方浮动搜索框（地名/坐标搜索）
- 点击地图放置标记，右侧浮窗显示坐标和最近 MERRA2 网格点距离
- 可切换卫星图/标准地图

**步骤 2 — 配置参数**
- 紧凑型表单，分组排布
- 高度选择使用 Tag/Chip 多选，支持自由数字输入（10～300 m），提供 10/50/80/100/120/140 m 快捷预设
- 地表覆盖类型下拉选择，配有图文说明辅助用户判断
- 时间范围使用年份滑块，实时显示对应数据年数
- 显示预估分析时长（根据年数和高度数估算）

**步骤 3 — 加载数据**
- 动态进度条 + 逐条加载日志（"正在获取 2015 年数据…"）
- 完成后自动跳转步骤 4

**步骤 4 — 分析总览**
- 6 个关键指标卡片：年均风速 / 年均 WPD / 主导风向 / Weibull k / Weibull c / 数据有效率
- 简版玫瑰图缩略图
- [查看详细分析] 按钮

**步骤 5 — 详细分析**
- 左侧导航菜单，依次：基础统计 / 玫瑰图 / Weibull / 风功率密度 / 风切变 / 湍流强度 / 极端风速（/ 多高度对比）
- 每个模块独立页，包含图表 + 关键数值卡片 + 说明文字
- 图表支持导出为 PNG

**步骤 6 — 生成报告**
- 填写封面信息的小表单
- 实时预览报告目录
- [生成 PDF] 按钮，生成完成后提供下载链接
